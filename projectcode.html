How To Make Images In HTML

To make images in HTML is very simple. You could use GIf's, JPG's, and PNG's as your images.

To put in an image you will first have to us the tag < img src= . The you will grab the link of the image you want to use for example "skyscraper.jpg". Now your line of code should look like this < img src="skyscraper.jpg"

You can leave you image looking like this if you like but there are ways to adjust the height and width of your image.

Height & Width (Style)

If you can't find the perfect sized image you need not a problem, just use the style tag you can change the size of the image to you liking.

To change the Height & Width of an image you have to use style= . One you have the tag Style= you must add in width & Height. So it should look something like this... style="width0px:; height:0px;">

Here you will enter your pixels to make it the size you want. style="width:250px;height:200px;">

Your Image is now the size that you want.
